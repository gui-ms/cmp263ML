# CMP263 - Aprendizagem de Máquina
Repositório contendo arquivos e códigos para o trabalho final da disciplina de Machine Learning (CMP263) da UFRGS.

Faça o download do dataset no link abaixo:
https://drive.google.com/drive/folders/1yHe16TJVvyvmVNviwvIiORXri4byKVNz?usp=sharing

## Instruções de Uso:

- Baixe o repositório do github;
- Baixe o dataset via google drive;
- Certifique-se de que o dataset se encontra na mesma pasta onde o notebook está OU
- Abra o notebook no google colab,e ao conectar ao ambiente de execução adicione o dataset baixado;
- Altere o caminho de acesso do dataset no código(procure por csv_path) de acordo com a opção selecionada acima (./artigo.csv, OU /content/artigo.csv, OU onde estiver o arquivo em relação ao notebook);
- Execute todas as células ou uma por uma;
- Aguarde.
